--> Email Sending using Python (SMTP)

-> Project Description

This project is about how to send emails using Python's built-in `smtplib` library.
It connects to Gmail SMTP server, authenticates using an app password, and sends a simple email message.

-> Features
->Send email using Python
->Secure connection using TLS
->Simple and beginner-friendly code


->How to Run
Replace credentials in code:

   `sender_email`
   `receiver_email`
   `app_password`
Run the script:
python send_email.py

Important Note 

 Use "App Password" not your Gmail password
 Enable 2-step verification in Gmail

Email will be sent successfully to the receiver.
